# Iris — универсальный игровой/PC ассистент
Goal:
Создать проект Iris — голосового AI-ассистента, который:
- Анализирует CS2 матч в реальном времени из GSI
- Общается живо, естественно и контекстно
- Поддерживает голосовой ввод/вывод
- Управляет базовыми функциями ПК (громкость, музыка и др.)
- Может интегрироваться со стрим-чатом (StreamElements)
- Имеет характер, юмор и гибкость

Requirements:

1) Архитектура
- Разделить проект на модули: core, gsi, speech, actions, stream
- Использовать async сервер для GSI
- Очистить репозиторий от логов/временных файлов

2) Модель общения
- Использовать языковую модель (Qwen3 или выше)
- Context manager — история разговора
- Не использовать статические скрипты

3) Speech
- ASR: VOSK/Whisper-tiny (бесплатно) ( он уже установлен vosk-model-ru-0.22)
- TTS: Coqui/Mozilla TTS
- Живой голос, приятный тембр

4) Game State Integration (CS2)
- Parse events → update context
- Fast updates (async)
- Separate player context

5) PC Integration
- Volume control
- Music controls
- App launch/close

6) Stream Support
- StreamElements API
- Chat read & respond

7) Personality
- Humor engine
- Custom tone

Constraints:
- Бесплатные инструменты и модели на старте
- Возможность масштабирования на платные сервисы
